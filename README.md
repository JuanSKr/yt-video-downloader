# YouTube Downloader

A simple program to download YouTube videos

## Installation
```bash
pip install pyside6
```
```bash
pip install pytube
```
```bash
pip install moviepy
```

## Usage
In the "Settings" section or by pressing Control + P you can set a path. Then enter the link to the video you want to download and choose the format.

> **Available formats**  
> - .mp4
> - .mp3
> - .wav
